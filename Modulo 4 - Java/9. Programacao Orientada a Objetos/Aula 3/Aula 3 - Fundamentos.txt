

-------------------------------------------------------------------------

Definição

"A Orientação a Objetos é um paradigma de análise, projeto e programação de sistemas de software  baseado na composição e interação entre diversas unidades de software chamadas de objetos." (Wikipedia)

Fundamentos

1. Abstração: "Processo pelo qual se isolam características de um objeto, considerando os que tenham em comum certos grupos de objetos."

2. Reuso: "Capacidade de criar novas unidades de código a partir de outras já existentes."

3. Encapsulamento: "Capacidade de esconder complexidades e proteger dados."
